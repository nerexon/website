What is it ?

Langpack is a module for translating Node.js Javascript applications. It's an easy-to-use system that's fast and efficient. It allows you to use JSON files to translate your application.

Documentation

Installation

You can install Langpack with the following command :

npm install langpack.js

Preparation

Before using Langpack, you must do some simple preparations. At first, you must create the directory and the files where the translations will be stored.

languages
    en.json
    fr.json

You must import the langpack.js module next and initializa the language manager class. The "LanguageManager" class require to specify the path of the directory where the translation JSON files are located.

const { LanguageManager } = require('langpack.js');
const manager = new LanguageManager('./languages');

The last step is to initialize the translator for every languages. You can use the "Translator" class to do so. You must specify the language.

const en = new manager.Translator('en');
const fr = new manager.Translator('fr');

Translation

You may now translate texts with the translator's "get" method. You must specify the key and the variables.

en.get("user.welcome", {username: "John"}); // "Welcome John!"

File exemple

There is a small and simple exemple of a translation JSON file.

{
    "user": {
        "welcome": "Welcome {user}!",
        "notification": "You received {count} notifications."
    }
}
