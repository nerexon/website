Our projects

KeychainDB

KeychainDB is a key-value database for Node.js applications. It allow to save data localy with breaking your head while configuring a server or a complexe database.

LangPack

LangPack or Language Pack is a language management system for Node.js applications. It allow translation of text into multiple languages by using simple JSON files. It is compatible with KeychainDB.

  • Learn more

Nerix

Nerix is a web server wrote in Go and Javascript (Node.js) made to be fast and simple. It allow the deployement of web applications in minutes. It supports a large variety of languages and frameworks.
