Qu'es-ce que c'est ?

Langpack est un module qui permet la traduction des applications Javascript Node.js. Il s'agit d'un système simple d'utilisation rapide et efficace. Il permet d'utiliser des fichiers JSON afin d'effectuer la traduction de votre application.

Documentation

Installation

Vous pouvez installer langpack avec la commande suivante :

npm install langpack.js

Préparation

Avant d'utiliser Langpack, vous devez effectuer quelques préparatifs simples. Vous devez dans un premier temps créer le dossier et les fichiers JSON pour stocker les traductions.

languages
    en.json
    fr.json

Vous devez ensuite importer le module langpack.js et initialiser le gestionnaire de langue. La classe « LanguageManager » demande de spécifier le répertoire où les fichiers de langue sont.

const { LanguageManager } = require('langpack.js');
const manager = new LanguageManager('./languages');

La dernière étape est d'initialiser les traducteurs pour chaque langues. Vous pouvez utiliser la classe « Translator » afin de le faire. Vous aller devoir spécifier la langue.

const en = new manager.Translator('en');
const fr = new manager.Translator('fr');

Traduction

Vous pouvez désormais traduire des textes avec la méthode « get » du traducteur. Vous devez spécifier la clée ainsi que les variables.

fr.get("user.welcome", {username: "John"}); // "Bienvenue John!"

Exemple de fichier

Voici un exemple de fichier de traduction json simple.

{
    "user": {
        "welcome": "Bienvenue {user}!",
        "notification": "Vous avez reçu {count} notifications."
    }
}
