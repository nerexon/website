Nos projets

KeychainDB

KeychainDB est une base de données « key-value » qui permet de stocker des données sur une application Node.js. Elle permet d'enregistrer des données localement sans devoir se casser la tête avec la configuration d'une base de donnée complexe.

LangPack

LangPack ou Language Pack est un système de gestion de langue pour les applications Node.js. Il permet de traduire les messages de l'application en différentes langues à l'aide de simple fichier JSON. Il est compatible avec KeychainDB.

  • En savoir plus

Nerix

Nerix est un serveur web écrit en Go et en Javascript (Node.js) conçu pour être simple et rapide. Il permet de déployer une application web en quelques minutes. Il support une large variété de langages de programmation et de « frameworks ».
