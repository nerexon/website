Qu'es-ce que c'est ?

KeychainDB est une base de donnée dites « key-value » pour les applications Node.js. C'est à dire, que vous pouvez stocker des données sous forme de clé et de valeur. Les données sont enregistrées dans la mémoire vive afin d'être le plus rapide possible. La base de donnée est locale afin d'avoir le moins de latence possible et d'être plus sécurisé. Avec son architecture modulaire, il est facile d'ajouter de nouvelles fonctionnalités à l'aide d'addons, permettant ainsi de personnaliser la base de donnée à vos besoins et de rajouter de nouvelles fonctionnalités.

Documentation principale

Installation

L'installation de KeychainDB est simple, vous pouvez l'installer via npm en utilisant la commande suivante :

npm install keychaindb

Importation et chargement

Avant de pouvoir utiliser votre base de donnée, vous devez importer le module est charger une nouvelle instance de la classe.

const { Database } = require("keychaindb");

const db = new Database();

Enregistrer une valeur

Pour enregistrer une valeur dans votre base de données, vous pouvez utiliser la méthode « set », vous devez alors spécifier une clée, une valeur et un délais en milliseconde si vous voulez que la valeur expire après un certain temps.
Dans l'exemple si dessous, une variable se nommant « Clef » ayant comme valeur « Valeur » expire après 5 secondes.

db.set("Clef", "Valeur", 5000);

Lire une valeur

Pour lire une valeur de votre base de données, vous pouvez utiliser la méthode « get ». Vous devez spécifier la clée que vous voulez lire. Vous allez ensuite récupérer une valeur .
Dans l'exemple si dessous, nous récupérons la valeur de la clée « Clef ».

db.get("Clef");
